# Official Knowledge Base Manifest

This directory is the shared knowledge base used by both `agent/` and `rag_chat_app/`.

## Scope

- Basel III final reforms and the output floor.
- Supervisory guidance on expected credit losses.
- IFRS 9 impairment / expected credit losses.
- Minimal China supervision overview.
- Short finance risk glossary.

## Source policy

Only official sources are used here. Prefer these websites when extending the corpus:

- [Bank for International Settlements (BIS)](https://www.bis.org/)
- [IFRS Foundation](https://www.ifrs.org/)
- [People's Bank of China](https://www.pbc.gov.cn/)
- [National Financial Regulatory Administration](https://www.nfra.gov.cn/)

## Working note

If a topic is not covered by an official source yet, do not invent content. Add the exact official document first, then regenerate this corpus.

## Frozen v1

The current mounted corpus is frozen for v1. Only fix factual errors or add a clearly necessary official source.
